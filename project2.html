ServoLink

ServoLink is an advanced prosthetic arm that captures real-time motion from a human hand and mimics it using servo-driven actuators. It utilizes motion sensors that uses camera to track the user's real finger and wrist movement, which are then replicated by a 3D-printed mechanical arm. Powered by a microcontroller.

Image

Features

  • Precise Control of Movement
  • Smooth and Responsive Action
  • Closed-Loop Feedback System
  • Closed-Loop Feedback System
  • High Torque and Compact Size
  • Energy Efficient

How it works

  • A microcontroller sends PWM (Pulse Width Modulation) signals to the servo motor from the camera.
  • The servo interprets the signal to move to a specific angular position.
  • An internal potentiometer provides real-time feedback to maintain precise positioning.
  • The servo motor adjusts continuously to hold the commanded position against external forces.
  • Smooth and accurate movement enables realistic actuation of linked parts like fingers or joints.

Components

  • 3D-Printed Prosthetic Hand
  • Arduino UNO Microcontroller
  • Servo Motors (SG90 / MG90S)
  • Breadboard (Mini)
  • Jumper Wires (Male-to-Male, Male-to-Female)
  • Scissors and Wire Stripping Tools
